
快速开始：

1. 创建并激活虚拟环境，然后安装依赖：

```powershell
# 在项目根目录运行
.\setup.ps1
```

2. 启动应用：

```powershell
.\run.ps1
```

3. 打开浏览器访问：http://localhost:5000

文件说明：

- `app.py`: Flask 应用入口
- `requirements.txt`: Python 依赖
- `templates/index.html`: 主页模板
- `static/css/style.css`: 简单样式
- `setup.ps1`: 创建虚拟环境并安装依赖（PowerShell）
- `run.ps1`: 激活虚拟环境并运行应用（PowerShell）
